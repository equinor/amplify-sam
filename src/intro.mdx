## Amplify SAM Storybook

This is the storybook for the `@equinor/amplify-sam` npm-package.

In here you will find components, hooks and providers that are made to communicate with the SAM system. This system provides an easy way to get features like: release notes, tutorials, feature-toggling, health/vulnerability checks.

This project is maintained and developed by and for the amplify team.

You can find the repository [here](https://github.com/equinor/amplify-sam).
